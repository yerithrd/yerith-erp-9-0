### <u>COMPILING 'YERITH-ERP-9.0'</u>

 1. SERVER CONFIGURATION
   use 'bash' script yr_compile_server.sh, e.g.:
   'yr_compile_server.sh 4 ae'

 2. STANDALONE CONFIGURATION
   use 'bash' script yerith-compile_yerith-erp-9-0-standalone.sh, e.g.:
   'yerith-compile_yerith-erp-9-0-standalone.sh -j4 -a -e'
